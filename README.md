# Shimeji with Chiaki Nanami Image Package

This project adds chiaki nanami image package to shimeji with corresponding action scripts.

Shimeji this project used is downloaded from this [website](https://kilkakon.com/shimeji/).

The original readme file is [here](./readme.txt).

## How to start

Firstly you need to install [java8](https://www.java.com/en/download/) on your computer.

Then left-click Shimeji-ee.jar twice you'll start the nanami mascot. 

## Image Generation

All the images are generated using [Google Gemini](https://gemini.google.com/app)  and [Sora](https://openai.com/zh-Hans-CN/sora/)

![](img/nanami/show.png)

## Supported Actions: 
* Walking
* Running
* Climbing edges
* Laying / creeping on the floor
* Falling and then standing up
* Idle motion 1: Standing with blinking
* Idle motion 2: Sleeping
* Idle motion 3: Playing games
* Windows Interactions: Grabing, pushing and finally throw the windows away

## Icon

The system tray icon of shimeji is replaced with [nanami](img/icon.png).

And you can generate a shortcut of Shimeji-ee.jar to the desktop and use the nanami00.ico to replace the desktop icon.
